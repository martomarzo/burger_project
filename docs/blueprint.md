# **App Name**: Burger Builder

## Core Features:

- Topping Management: Allow the user to add available burger toppings with a name, and persistent storage of those toppings for creating the burgers later on.
- Burger Creation: Allow the user to create custom burgers for people by assigning a name to the burger (the name of the person eating it).
- Burger Visualization: Visually represent the burger being built, displaying the toppings selected, and the name assigned to the burger.
- Burger List: Persist and manage the current list of built burgers

## Style Guidelines:

- Primary color: Earthy brown (#A68B64) to evoke a burger vibe.
- Secondary colors: Yellow (#FFDA63) and red (#FF6B6B) for topping highlights.
- Accent: Teal (#76D4FF) for CTAs and interactive elements.
- Clean and modular layout for easy navigation and clarity.
- Use delicious looking burger and topping icons to aid visual recognition.
- Subtle animations for topping selection, burger creation, and completion.
